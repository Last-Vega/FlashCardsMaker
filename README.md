# Features

* ログイン機能を用いて自分オリジナルの単語帳を作成可能
* 単語帳が備えるべき索引機能も作成
* 連続正解数と連続不正解数をカウント

# Usage

* ログイン->単語登録->インプット->アウトプット->不正解箇所の復習　を繰り返す
